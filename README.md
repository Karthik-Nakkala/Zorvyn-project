## 📋 Project Overview
Clean, modern, and fully interactive **Personal Finance Dashboard** built for Zorvyn FinTech.

Users can:
- View real-time financial summary cards
- Track balance trend over last 6 months
- See spending breakdown with beautiful pie chart
- Manage transactions (Add / Delete / Search / Sort)
- Role-based access (Admin vs Viewer)
- Smart AI-like insights
- Export all transactions as CSV
- Data persists using LocalStorage

## 🛠️ Tech Stack
- **React 18** + Vite
- **Redux Toolkit** (State Management)
- **Tailwind CSS v4** (Styling)
- **Recharts** (Charts)
- **TanStack Table v8** (Transactions table with sorting & filtering)
- **Lucide React** (Icons)
- LocalStorage for persistence

## ✨ Key Features
- **Summary Cards** – Total Balance, Income, Expenses, Savings Rate
- **Dynamic Charts** – Balance Trend (Line) + Spending Breakdown (Pie)
- **Fully Functional Transactions Table** – Search, Sort, Filter
- **Role Simulation** – Admin can Add/Delete, Viewer can only view
- **Smart Insights** – Highest spending category + Monthly comparison
- **CSV Export** – One-click download
- **LocalStorage Persistence** – Data remains after refresh
- **Responsive & Modern Dark UI**
